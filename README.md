# dot

This is my dotfiles repository.

## Installation

### dependencies

```sh
dnf install git stow
```

## Clone the repo

```sh
git clone https://github.com/clobrano/dot .dot
```

## Linking

```sh
cd ~/.dot
stow .
```

(Thanks to https://www.youtube.com/watch?v=y6XCebnB9gs for the introduction to Stow)
